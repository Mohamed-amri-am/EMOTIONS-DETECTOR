# EMOTIONS-DETECTOR

Face Analysis Application
Overview
A comprehensive facial analysis application that uses multiple machine learning models to detect and analyze facial features in real-time or from images. The application supports age, gender, ethnicity, emotion, and color analysis with a modern cyberpunk-themed interface.

Features
Core Functionalities
Real-time facial analysis using webcam feed

Image analysis from uploaded files

Multi-model support with interchangeable prediction models

Multiple facial attribute detection:

Age estimation

Gender classification

Ethnicity recognition

Emotion detection (7 basic emotions)

Dominant color extraction

Display Features
Real-time video streaming with face detection

Facial landmarks visualization using MediaPipe

Customizable display options

Results highlighting with colored boxes

Supported Models
Age Detection
DeepFace: Comprehensive facial analysis framework

AgeGenderDeep: Specialized age and gender detection

Caffe: Berkeley AI Research model

YU4U: Age-only ResNet50 model

Gender Detection
DeepFace

AgeGenderDeep

Ethnicity Detection
DeepFace

VGG-Face: Oxford Visual Geometry Group model

Emotion Detection
DeepFace

FER (Facial Emotion Recognition): Trained on FER-2013 dataset
